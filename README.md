# ABC 精密 · 官網與營運週報（匿名化 DEMO）

本 repo 以 GitHub Pages 發佈兩個自包含、離線可用的 HTML 頁面。

| 頁面 | 路徑 | 線上網址 |
|---|---|---|
| **官網 Landing Page**（精密淨白風格、含 Hero Banner、SEO） | `index.html` | https://yuinca.github.io/ERP-DEMO/ |
| **營運週報儀表板**（業績／進貨／財務＋AI 洞見） | `report/index.html` | https://yuinca.github.io/ERP-DEMO/report/ |

## 官網
光電・電子・自動化的精密製造廠形象官網，採「精密淨白」工程美學，含製造能力、製程流程、應用產業、品質認證與 FAQ；具備 `Organization` 與 `FAQPage` 結構化資料、Open Graph 與語意化 HTML。

## 營運週報
每週業績、進貨、財務三分頁，每個關鍵指標附「AI 洞見」（問題分析／問題來源／建議對策），右上可切換週次即時重算。

> 所有公司名、數字、聯絡資訊均為**匿名化測試／佔位資料**，不含任何真實營運資料，亦不含 ERP 原始 Excel 檔。
