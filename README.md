# ERP-DEMO · 營運週報

ABC 公司（匿名化 DEMO／測試資料）的營運週報儀表板，單一自包含 HTML、離線可用。

- **業績週報** — 銷貨額、客戶排行、業務員業績、近 12 週趨勢
- **進貨週報** — 進貨額、供應商排行、趨勢
- **當週財務報告** — 收入／進貨成本／毛利、應付帳款、收入 vs 成本趨勢

右上角可切換週次，切換即時重算。

> 本頁所有數字皆為**匿名化測試資料**（公司名、表名、日期均經過去識別化處理），不含任何真實營運資料，亦不含 ERP 原始 Excel 檔。

線上版（GitHub Pages）：開啟 repo 根目錄的 `index.html`。
